# 🔗 Как показать сайт покупателю по ссылке (самый простой способ)

Сайт работает у тебя на ПК, а туннель даёт временный публичный адрес. ПК должен быть
включён во время показа. Регистрация не нужна.

## Cloudflare Tunnel (рекомендую — чисто, без заглушек)

1. **Запусти сайт.** В папке проекта:
   ```
   npm start
   ```
   Окно не закрывай.

2. **Скачай cloudflared** (один файл):
   - https://github.com/cloudflare/cloudflared/releases/latest
   - файл **`cloudflared-windows-amd64.exe`**
   - переименуй в **`cloudflared.exe`**, положи в папку `lavashok`.

3. **Открой второе окно PowerShell** и запусти туннель:
   ```
   cd C:\Users\mzolo\Desktop\claude\lavashok
   .\cloudflared.exe tunnel --url http://localhost:3000
   ```

4. **Скопируй ссылку** из вывода (вида `https://....trycloudflare.com`) и отправь покупателю.
   - Сайт: `https://....trycloudflare.com/`
   - Админка: `.../admin`
   - Кухня: `.../kitchen`

5. **После показа** закрой оба окна PowerShell — ссылка перестанет работать.

## Быстрый вариант без скачивания

Во втором окне PowerShell:
```
npx localtunnel --port 3000
```
Даст ссылку `https://xxxx.loca.lt`. Минус: при первом заходе покажет страницу-предупреждение.

## Если покупатель рядом

Туннель не нужен: `npm start` → открой `http://localhost:3000` на весь экран.
На телефон в той же Wi-Fi — по IP компьютера (см. `ИНСТРУКЦИЯ.md`).

---

Туннель — только для показа. Чтобы сайт работал постоянно без твоего ПК — облако или VPS
(см. `ТАЙМВЕБ.md` и `DEPLOY.md`).
